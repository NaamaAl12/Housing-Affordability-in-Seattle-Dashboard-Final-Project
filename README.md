## Project Description  

The Seattle Housing Affordability Dashboard is an interactive smart dashboard that explores rent burden, household income, and Mandatory Housing Affordability (MHA) zoning in Seattle from 2006–2022.

The project combines a choropleth map and coordinated charts to help users understand how rent burden varies across time, neighborhoods, and demographic groups.

Users can:
- Hover over census tracts to see detailed popup information  
- Filter data by year using a timeline slider  
- Compare rent burden by demographic group  
- Toggle map layers to view MHA zoning tiers  
- View coordinated charts that update dynamically  

---

## Project Goal  

The goal of this project is to:

- Identify spatial concentrations of renter cost burden in Seattle  
- Compare rent burden with median household income  
- Explore renter income distribution across neighborhoods  
- Provide spatial context through mapping of MHA zoning tiers  
- Allow interactive exploration through linked maps and charts  

Through interactive visualization, users can better understand where housing affordability challenges are most concentrated and how urban growth patterns may relate to these disparities.
